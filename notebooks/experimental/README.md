# 🧪 Experimental Notebooks

이 디렉토리는 시도하지 못한 실험적 노트북들을 포함합니다.

## 📝 내용

### 시도하지 못한 실험들

1. **영어 번역 실험**

   - 데이터를 영어로 번역하여 학습
   - Qwen 모델의 영어 최적화 활용
   - 파일 위치: `data/translated/`

2. **프롬프트 체계화**

   - 더 구체적이고 체계적인 시스템 프롬프트
   - 규칙 기반 출력 형식

3. **더 큰 모델 사용**

   - 7B 또는 그 이상의 모델
   - 성능 향상 기대

4. **에폭 수 증가**

   - Epoch = 3~5
   - 충분한 학습으로 언더피팅 방지

5. **데이터 증강**
   - 다양한 증강 기법 적용

## ⚠️ 주의사항

이 디렉토리의 노트북들은:

- 시간 제약으로 실행하지 못한 실험들
- 향후 시도할 아이디어들
- 완성되지 않은 코드들
