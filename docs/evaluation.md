## 📊 평가 메트릭

### 정확도 (Accuracy)

**정의**:

- 전체 테스트 샘플 중 올바른 답변 비율
- 범위: 0.0 ~ 1.0

**계산식**:

```
정확도 = (정답 개수) / (전체 샘플 수)
```

### 평가 예시

| 정답 | 예측 | 정확도 |
| ---- | ---- | ------ |
| a    | a    | ✅     |
| b    | a    | ❌     |
| c    | c    | ✅     |
| d    | d    | ✅     |
| a    | b    | ❌     |

정확도 = 3/5 = 0.6
