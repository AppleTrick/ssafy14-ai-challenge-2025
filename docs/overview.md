## 🎯 미션

### 목표

Qwen VQA 모델의 학습 및 추론 성능 최적화

### 태스크

- Visual Question Answering (VQA)
- 주어진 이미지와 질문에 대해 올바른 답변 선택
- 답변은 a, b, c, d 중 하나

### 데이터

- **학습 데이터**: train.csv (3,887 samples)
- **테스트 데이터**: test.csv
- **제출 형식**: CSV (image_id, answer)
