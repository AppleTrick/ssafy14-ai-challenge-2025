## 📊 제출 파일 형식

### CSV 형식

```csv
id,answer
test_0001,a
test_0002,b
test_0003,c
...
```

### 주요 규칙

1. **헤더 필수**: `id,answer`
2. **답변 형식**: 소문자 a, b, c, d 중 하나
3. **파일 인코딩**: UTF-8
4. **행 순서**: ID 순서대로 정렬
